# easy-translate

快速下载文件到目标服务器的 Edge 浏览器插件

## ✨ 特性

- 🚀 **快速文件传输**: 一键下载文件到目标服务器
- 🔒 **安全传输**: 支持HTTPS加密传输
- 🎯 **简单易用**: 简洁的用户界面，操作便捷
- 🔧 **高度可配置**: 支持自定义服务器配置
- 📊 **传输监控**: 实时显示传输进度和状态

## 🚀 快速开始

### 安装

1. 下载插件文件到本地
2. 打开Edge浏览器，进入扩展管理页面
3. 开启开发者模式
4. 点击"加载解压缩的扩展"
5. 选择插件目录

### 使用

1. 在浏览器中点击插件图标
2. 配置目标服务器信息
3. 选择要下载的文件
4. 点击下载按钮开始传输

## 🛠️ 开发

### 环境要求

- Node.js >= 14.0.0
- npm >= 6.0.0

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发模式
npm run dev

# 构建生产版本
npm run build

# 清理构建文件
npm run clean
```
